OPESP Website - Image Directory
================================

This directory contains all images for the OPESP website.

Structure:
- images/logo.png         : OPESP main logo (used in navbar)
- images/favicon.ico      : Favicon ICO format
- images/favicon.png      : Favicon PNG format
- images/hero-estaiada.jpg: Hero section background image

- images/membros/         : Member photos
  membro-01.jpg through membro-12.jpg (or more as needed)

- images/eventos/         : Event photos organized by event
- images/premio/          : Top Business award ceremony photos
  by edition year (2014-2023)

Image Guidelines:
- Member photos: 90x90px minimum, square format
- Hero image: 1920x1080px recommended
- Gallery photos: 800x600px minimum, landscape format
- Logo: SVG or PNG with transparency, min 200px wide
